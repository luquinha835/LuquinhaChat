# 💬 Luquinha Chat

Um chat em tempo real desenvolvido em **HTML**, **CSS** e **JavaScript**, utilizando o **Firebase Cloud Firestore** como banco de dados.

## ✨ Recursos

- 💬 Chat em tempo real
- 🔥 Firebase Cloud Firestore
- 👤 Nome do usuário
- ⌨️ Envio de mensagens com Enter
- 📱 Interface responsiva
- 🌙 Tema escuro
- ⚡ Atualização automática das mensagens

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript (ES Modules)
- Firebase
- Cloud Firestore

## 🚀 Como usar

### 1. Clone o repositório

```bash
git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
```

### 2. Abra o projeto

Basta abrir o arquivo **index.html** em um navegador ou publicar o projeto no **GitHub Pages**.

### 3. Configure o Firebase

No arquivo `index.html`, localize o objeto:

```javascript
const firebaseConfig = {
    ...
}
```

Substitua pelos dados do seu projeto Firebase, caso necessário.
